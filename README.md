# KeyBox  
###这是按照学长的密钥盒子做的，做了钥匙的添加，展示，编辑等功能，里面用了SQLite数据库及SharedPreferences存储，主界面是material风格，做了一个drawerLayout，使用cardView来显示密匙，并可以在设置中设置密码是否显现。
